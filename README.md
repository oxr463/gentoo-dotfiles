# .gentoo

_Miscellaneous files to assist in maintaining Gentoo Linux._

## Instructions

### Setup .gentoo repository

```sh
git clone https://github.com/oxr463/gentoo-dotfiles.git ~/.gentoo
cd ~/.gentoo
docker-compose build
docker-compose run gentoo-sdk /bin/sh
```
